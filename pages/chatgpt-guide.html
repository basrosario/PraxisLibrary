What is ChatGPT?

OpenAI's conversational AI assistant

ChatGPT is a large language model developed by OpenAI that can understand and generate human-like text. It's designed for conversational interactions and can help with writing, analysis, coding, brainstorming, and many other tasks.

ChatGPT maintains conversation context, meaning it remembers what you've discussed within a single chat session. This makes it ideal for multi-turn conversations where you refine ideas together.

ChatGPT Projects

Organize your work with persistent context

Persistent Memory

Projects remember context across multiple conversations. Upload reference documents, set instructions, and ChatGPT maintains that knowledge throughout.

Custom Instructions

Set project-specific instructions that apply to every conversation. Define your role, preferred output style, and constraints once.

File Uploads

Upload documents, spreadsheets, and code files. ChatGPT can reference, analyze, and work with your uploaded content.

Organized Workspace

Keep related conversations grouped together. Great for ongoing projects, research, or any work that spans multiple sessions.

Using Prompting Methods with ChatGPT

Apply what you've learned to get better results

01

CRISP for Quick Tasks

Use CRISP (Context, Role, Instructions, Specifics, Parameters) for straightforward requests. ChatGPT responds well to structured prompts.

02

Custom Instructions = Context

ChatGPT's custom instructions feature is perfect for the "Context" and "Role" parts of your prompts. Set them once, use them always.

03

COSTAR for Content

When creating content, use COSTAR to specify audience and tone. ChatGPT can adapt its writing style significantly based on these inputs.

04

Flipped Interaction

Ask ChatGPT to interview you first. It excels at asking clarifying questions that help it give more personalized responses.

Best Practices

Start with context, then request

Before asking ChatGPT to do something, give it the background it needs. "I'm planning a community event for 50 people with a $500 budget. Help me create an agenda." works better than "Create an event agenda."

Be specific about format

ChatGPT can output in many formats: bullet points, tables, numbered lists, paragraphs, code blocks. Tell it what you want: "Format as a table with three columns" or "Give me 5 bullet points."

Iterate and refine

Don't expect perfection on the first try. Say "make it shorter," "add more examples," or "make the tone more casual." ChatGPT remembers the conversation and can refine its output.

Use examples when possible

If you have a specific style or format in mind, show ChatGPT an example. "Write something like this: [example]" is more effective than describing what you want.

Verify important information

ChatGPT can make mistakes, especially with facts, dates, and current events. Always verify important information from reliable sources before using it.
