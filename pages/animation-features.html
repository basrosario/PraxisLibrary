Animation & Features

A comprehensive reference showcasing all available animations, components, and visual features for the Praxis Library.

Basic Animations

Fade-In Animations

Elements fade in as they enter the viewport

fade-in-up

Elements fade in while sliding up from below. Great for cards and content blocks.

fade-in

Simple fade in animation. Elements appear smoothly without movement.

Stagger Ready

Add stagger-children class to parent for sequential animations.
