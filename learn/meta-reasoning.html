Meta-Reasoning Prompting

Think about thinking—choose the right approach first.

The Concept

Reasoning About Reasoning

Meta-Reasoning prompts the model to first analyze what type of problem it's facing and select an appropriate reasoning strategy before attempting to solve it. This ensures the right tool for the job.

Key Questions

What kind of problem is this? What strategies would work best? What could go wrong? Then proceed with the chosen approach.

The Pattern

"Before solving, analyze: What type of problem is this? What's the best reasoning approach for this type?"

Analysis

Strategy Selection

Problem Type

Is this math, logic, creative, factual, or multi-step? Different types need different approaches.

Complexity

How complex is the problem? Simple questions don't need elaborate reasoning chains.
